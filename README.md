# Banco com React

## Informação
Este repositório contém uma simulação de um banco simples escrito em React utilizando uma estrutura forte com <strong>páginas</strong>, <strong>hooks</strong>, <strong>utils</strong> e <strong>css</strong> organizado.

## Como funciona
De forma simples, o site funciona com uma simples, mas clara, interface que fornece opções para a manipulação simulada dos dados. Mantenha em mente que nada é salvo.

## Impacto
Demonstra como uma boa estrutura faz um bom site e amplifica nossa visão para projetos maiores, desenvolvi esse site com o intuito de mostrar minhas capacidades em React manipulando o DOM, estrutura JSON, estruturação e modularização.
